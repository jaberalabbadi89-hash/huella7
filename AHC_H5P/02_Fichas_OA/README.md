# 02_Fichas_OA

Una **ficha por OA** (`OA-XX_Ficha.md`). Hay dos tipos según el estado del objeto:

- **OA-01 a OA-10 — fichas documentales (terminados).** Estructura oficial de **8 bloques**, basada en el
  contenido REAL del paquete `.h5p` ya construido: Identificación · Diseño pedagógico · Contenido ·
  Configuración H5P · Accesibilidad · Moodle · Reutilización · QA y mejora.
- **OA-11 a OA-20 — fichas de diseño (prototipados, pendientes de construcción).** Definen el OA listo para
  montar en el editor H5P (consigna, ítems, feedback, configuración prevista, accesibilidad y pasos de
  construcción). El `.h5p` se autoría y documenta como ficha de 8 bloques **después** de construirlo.
