# Ficha Técnica - OA-02

| Campo | Detalles |
| :--- | :--- |
| **Código OA** | OA-02 |
| **Nombre** | Alcances de Emisiones (Scope 1, 2, 3) |
| **Competencia o habilidad** | Capacidad para categorizar correctamente las fuentes de emisiones corporativas según los estándares internacionales. |
| **Objetivo de aprendizaje** | Clasificar las fuentes de emisiones directas e indirectas en sus respectivos alcances. |
| **Tema** | Medición y Reducción de Emisiones |
| **Descripción** | Actividad interactiva donde el usuario debe arrastrar diferentes ejemplos de emisiones y soltarlos en la zona del alcance correspondiente. |
| **Tipo de actividad H5P** | Drag and Drop |
| **Nivel Bloom** | Analizar / Clasificar |
| **Duración** | 3 - 5 minutos |
| **Instrucciones** | Arrastra cada elemento de texto y suéltalo en la caja del Alcance (1, 2 o 3) que le corresponda. Pulsa 'Comprobar' al finalizar. |
| **Feedback** | Retroalimentación visual inmediata (verde/rojo) al pulsar comprobar, con opción de reintentar. |
| **Evidencia de aprendizaje** | Puntuación obtenida al colocar correctamente los 6 elementos. |
| **Criterios de accesibilidad** | Contraste de colores verificado. Fuentes legibles. (WCAG 2.1 AA) |
| **Compatibilidad Moodle** | 100% compatible mediante plugin H5P nativo. |
| **Orientación de reutilización** | Se pueden modificar los textos de las etiquetas para adaptar la actividad a otros sectores (ej. emisiones de un hospital o escuela). |
