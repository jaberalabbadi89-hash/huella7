# Ficha Técnica - OA-01

| Campo | Detalles |
| :--- | :--- |
| **Código OA** | OA-01 |
| **Nombre** | Conceptos Climáticos Básicos (CO2e, Net Zero) |
| **Competencia o habilidad** | Capacidad para identificar y recordar la terminología fundamental del cambio climático y la sostenibilidad. |
| **Objetivo de aprendizaje** | Recordar los conceptos clave del cambio climático, la huella de carbono y la neutralidad de carbono. |
| **Tema** | Concienciación Climática Básica |
| **Descripción** | Actividad de tarjetas didácticas de doble cara. El usuario lee un término o pregunta en el anverso y gira la tarjeta para leer la definición correcta en el reverso. |
| **Tipo de actividad H5P** | Dialog Cards |
| **Nivel Bloom** | Recordar |
| **Duración** | 3 - 5 minutos |
| **Instrucciones** | Lee la pregunta en la tarjeta y piensa en la respuesta. Luego, pulsa "Girar" para comprobar si tu definición es correcta. Pasa a la siguiente tarjeta cuando termines. |
| **Feedback** | Autoevaluación del propio estudiante tras girar la tarjeta. |
| **Evidencia de aprendizaje** | Finalización de la visualización del mazo de tarjetas (completado). |
| **Criterios de accesibilidad** | Textos grandes y claros. Navegación compatible con teclado. (WCAG 2.1 AA) |
| **Compatibilidad Moodle** | 100% compatible mediante plugin H5P nativo. |
| **Orientación de reutilización** | Las tarjetas pueden ser editadas fácilmente para añadir nuevos conceptos o traducirlas a otros idiomas (ej. Catalán). |
